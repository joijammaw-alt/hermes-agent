## 🛠️ ส่วนที่ 1: โปรแกรมพื้นฐานที่ต้องติดตั้งในเครื่อง (Prerequisites)
ก่อนเริ่มเรียนรู้ ให้คุณครูพาเด็ก ๆ ติดตั้ง 4 โปรแกรมหลักนี้ในเครื่องคอมพิวเตอร์ก่อนครับ:

1. **Python (เวอร์ชัน 3.10 ขึ้นไป) 🐍**: เปรียบเสมือนเครื่องยนต์หลักที่ทำให้ตัว Brain/Agent ทำงานได้
   * *ดาวน์โหลดที่*: [python.org](https://www.python.org/downloads/) (ตอนติดตั้งต้องติ๊กเลือก **"Add Python to PATH"** ทุกครั้ง!)
2. **Node.js (LTS Version) 🟢**: เพื่อรันระบบจำลองหน้าเว็บไซต์ Astro
   * *ดาวน์โหลดที่*: [nodejs.org](https://nodejs.org/)
3. **Obsidian 📄**: โปรแกรมเก็บรวบรวมความรู้และสมุดบันทึกแบบเชื่อมโยงเครือข่ายความรู้
   * *ดาวน์โหลดที่*: [obsidian.md](https://obsidian.md/)
4. **Local by Flywheel 🌐**: โปรแกรมจำลองระบบเว็บไซต์บล็อก WordPress บนคอมพิวเตอร์ของเราเอง
   * *ดาวน์โหลดที่*: [localwp.com](https://localwp.com/)

## 📂 ส่วนที่ 2: ขั้นตอนการตั้งค่าโปรเจกต์ทีละสเต็ป (Step-by-Step Tutorial)

### 📁 ขั้นตอนที่ 1: สร้างโฟลเดอร์โปรเจกต์ และเตรียมสภาพแวดล้อม Python
ให้เด็ก ๆ เปิด Terminal หรือ PowerShell แล้วรันคำสั่งเหล่านี้ทีละบรรทัดเพื่อสร้างบ้านหลังแรกให้โปรเจกต์:

1. **สร้างโฟลเดอร์และเข้าไปด้านใน**:
   ```bash
   mkdir hermes-agent
   cd hermes-agent
   ```
2. **สร้างพื้นที่จำลองส่วนตัว (Virtual Environment - `venv`)** เพื่อให้โปรแกรมต่าง ๆ ไม่ตีกันในเครื่อง:
   ```bash
   python -m venv venv
   ```
3. **เปิดใช้งาน `venv` (สำหรับ Windows)**:
   ```bash
   .\venv\Scripts\activate
   ```
4. **สร้างไฟล์ [requirements.txt](file:///d:/hermes-agent/requirements.txt) และใส่แพ็กเกจที่ต้องติดตั้ง**:
   ```text
   crewai>=0.28.0
   langchain-openai>=0.1.0
   python-dotenv>=1.0.0
   beautifulsoup4>=4.12.0
   requests>=2.31.0
   ```
5. **สั่งติดตั้งแพ็กเกจ**:
   ```bash
   pip install -r requirements.txt
   ```

---

### 🔑 ขั้นตอนที่ 2: การสร้างไฟล์ตัวแปรลับ (ตั้งค่าไฟล์ `.env`)
อธิบายให้เด็ก ๆ ฟังว่า: *"ไฟล์ .env เปรียบเหมือนกระเป๋าเก็บกุญแจลับต่าง ๆ ที่ห้ามให้ใครรู้เด็ดขาด"*
1. สร้างไฟล์ชื่อ **`.env`** ไว้ที่โฟลเดอร์หลักของโปรเจกต์
2. นำข้อความการตั้งค่าด้านล่างนี้ไปใส่ไว้ในไฟล์:
   ```env
   # คีย์สำหรับให้ AI Agent ใช้คิดประมวลผล (สมัครฟรีที่: platform.deepseek.com)
   DEEPSEEK_API_KEY=ใส่กุญแจคีย์ของน้องๆตรงนี้
   
   # คอนฟิกการส่งข้อมูลไปบล็อก WordPress ปลายทาง
   WP_URL=http://localhost:10004
   WP_USER=Music2412
   WP_PASS=T3Pf 1fYT 0UGb GYgj neln 00Jx
   ```

---

### 📂 ขั้นตอนที่ 3: ตั้งค่าโฟลเดอร์รับเอกสาร (Obsidian Vault)
1. เปิดโปรแกรม **Obsidian** บนคอมพิวเตอร์
2. กดเลือก **"Create new vault"** (สร้างคลังความรู้ใหม่)
3. ตั้งชื่อว่า `Obsidian_Vault` และเลือกตำแหน่งจัดเก็บไว้ภายใต้โฟลเดอร์โปรเจกต์หลัก (`d:\hermes-agent\Obsidian_Vault`)
4. เมื่อหุ่นยนต์ทำงานเสร็จ ข่าวสารที่ขูดมาจะไปโผล่ในโปรแกรม Obsidian โดยอัตโนมัติ!

---

### 🌐 ขั้นตอนที่ 4: สตาร์ทเปิดระบบเว็บบล็อก (WordPress Local Site)
1. เปิดโปรแกรม **Local by Flywheel**
2. กดปุ่มบวก **"+"** ที่มุมล่างซ้าย เพื่อสร้างเว็บไซต์ใหม่ (ตั้งชื่อว่า `hermes-test`)
3. เลือกการตั้งค่าเริ่มต้น และกำหนด Username/Password
4. เข้าหน้าหลังบ้านของ WordPress ไปที่ **Users -> Profile** เลื่อนลงไปข้างล่างสุด กดปุ่ม **"Add New Application Password"** ตั้งชื่อแอปว่า `hermes` 
5. นำรหัสผ่านที่ระบบแสดงให้ (รหัสผ่าน 24 หลัก) ไปใส่ไว้ในช่อง `WP_PASS` ในไฟล์ `.env` ข้างต้น

---

### 🎨 ขั้นตอนที่ 5: ตั้งค่าโครงสร้างเว็บไซต์ Astro
1. ติดตั้งโครงสร้างพื้นฐานโดยเปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วพิมพ์:
   *สร้างไฟล์ `package.json` ปลายทางเพื่อรับแพ็กเกจของเว็บ:*
   ```json
   {
     "name": "hermes-watchara-astro",
     "type": "module",
     "version": "1.0.0",
     "scripts": {
       "dev": "astro dev --host"
     },
     "dependencies": {
       "astro": "^4.16.0"
     }
   }
   ```
2. รันคำสั่งดาวน์โหลดระบบจำลองหน้าเว็บจริง:
   ```bash
   npm install
   ```
3. เมื่อติดตั้งเสร็จ โฟลเดอร์ `src/pages/index.astro` จะถูก AI นำโค้ดหน้าเว็บสุดล้ำไปเขียนแต่งไว้ให้ทันที!

## 🧠 ส่วนที่ 3: การรันและชื่นชมผลงานด้วยตัวเอง! 🚀

คุณครูสามารถสอนให้เด็ก ๆ ควบคุมและสังเกตการทำหน้าที่ของหุ่นยนต์ AI ทั้ง 3 ขั้นตอนดังนี้ครับ:

### 1) สั่งให้หุ่นยนต์ AI ออกปฏิบัติการ 🤖
พิมพ์คำสั่งรันระบบสมองหลักลงใน Terminal:
```bash
venv\Scripts\python.exe main.py
```
* **สิ่งที่เด็ก ๆ จะได้สังเกตเห็น**:
  1. หุ่นยนต์จะเริ่มทำการตรวจสอบความพร้อมของระบบและสวมสมอง DeepSeek
  2. หุ่นยนต์เริ่มรัน **Task 1**: เรียกใช้โปรแกรมขูดข่าว (`scrape_watchara_school`) และเริ่มเขียนไฟล์ Markdown ปรากฏในคลังความรู้ Obsidian ทันที!
  3. หุ่นยนต์ทำต่อที่ **Task 2**: วิเคราะห์และเลือกสรร **3 ข่าวที่น่าสนใจและล้ำที่สุด** แล้วส่งโพสต์ขึ้นไปที่ WordPress โดยอัตโนมัติ (เด็ก ๆ สามารถเข้าไปเช็กหน้าบล็อกจริงได้เลย)
  4. หุ่นยนต์รัน **Task 3**: นำข่าวทั้งหมดมาสร้างสรรค์ ตกแต่งเขียนทับโค้ดในโปรเจกต์ Astro ด้วยสไตล์สุดเท่ Futuristic Dark Mode

### 2) เปิดรันจำลองเว็บไซต์เพื่อดูความงดงาม 🖥️✨
พิมพ์คำสั่งนี้เพื่อเปิดให้เว็บไซต์ของเรามีชีวิตและรันระบบจำลองออนไลน์ขึ้นในเครื่อง:
```bash
npm run dev
```
* **สิ่งที่เด็ก ๆ จะได้ชม**:
  * ให้เปิด Google Chrome หรือเว็บเบราว์เซอร์ แล้วเปิดหน้าเว็บไปที่: **`http://localhost:4321/`**
  * เด็ก ๆ จะได้เพลิดเพลินกับผลงานหน้าแรกโรงเรียนวัชระโฉมใหม่ที่มีปุ่มกดเลือกคัดกรองข่าวสารแบบไดนามิก ตัวหนังสือสีนีออนเรืองแสงสะดุดตา แอนิเมชันน่าตื่นเต้น และข้อมูลที่เชื่อมโยงกับบล็อก WordPress ได้อย่างสมบูรณ์แบบครับ!
